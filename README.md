## .Net Core (Commands)

### Check version
```shell
dotnet --version
```

### List the templates
```shell
dotnet new list
```

### Generate a webapi with the name "api"
```shell
dotnet new webapi -n api
```

### RUN .Net app
```shell
dotnet run
```
